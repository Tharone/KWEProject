In this folder test_data, there are 27 .txt files. 
Each .txt file maps to an OCR output of a picture.
To identify which file maps to which picture's OCR output, kindly refer to the naming convention below:

- "T1.txt" to "T23.txt" -> OCR outputs of pictures "Cont1" to "Cont23"
- "T24.txt" -> OCR output of "Cover5"
- "T25.txt" -> OCR output of "Cover7"
- "T26.txt" -> OCR output of "Cover9"
- "T27.txt" -> OCR output of "Cover20"
